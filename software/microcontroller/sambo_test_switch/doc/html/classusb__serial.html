void usb_serial::rx_callback ( void  )

USB Serial RX callback function. This function should be called every time data is received on the USB Serial line so that the data is added to the RX buffer correctly. The usb stack provides a place to call a notify function on rx and this function is indirectly called by that function. See usb_serial_rx_notify at the top of this file for the wrapper function that calls this function.
